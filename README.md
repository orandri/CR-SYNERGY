# CR-SYNERGY

Site statique HTML/CSS/JavaScript, sans build et sans dépendance Node.

## Déploiement GitHub Pages

1. Pousser le projet sur GitHub.
2. Dans le dépôt GitHub, aller dans `Settings` > `Pages`.
3. Choisir la branche de déploiement, par exemple `main`.
4. Choisir le dossier racine `/`.
5. Enregistrer.

Le site démarre directement sur `index.html`.
